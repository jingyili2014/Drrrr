== Description ==

Kheera is a clean and modern WordPress Blog & Shop theme, with an elegant, carefully crafted pixel perfect design.
This powerful, responsive blog theme is ideal for your personal or magazine-style blog.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
